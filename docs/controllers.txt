
Контроллеры - менеджер сбора данных определённой структуры

Конроллеры ресурсов - созданы для обработки запросов api запросов
- реализуют систему CRUD

https://laravel.su/docs/8.x/controllers

1 создаём контроллер ресурсов
php artisan make:controller PhotoController --resource
- в нём будут описаны стандартные действия реализующие систему CRUD
2 создаём маршрут, через специальный метод
Route::resource('/posts', 'PostController');
- метод подключает систему построения url для обращений к методам контроллера
- название параметра обозначающего конкретный элемент можно изменить
https://skr.sh/sGhnEppJB0a
3 Реализуем фун-ал обращения к методам контроллера
- названия маршрутов и необходимых параметров берём из таблички
- маршруты вставляем в ссылки в методы формы
- передаём в параметры маршрута необходимые значения
4 реализуем логику методов контроллера

Действия контроллера:
posts.index - список элементов
posts.create - форма добавления нового элемента
posts.store - сохранение нового элемента
posts.show - просмотр элемента детально
posts.edit - форма изменения элемента
posts.update - сохранение изменений элемента
posts.destroy - удаление элемента
