Seeding - массовое заполнение бд тестовыми данными
https://laravel.su/docs/7.x/seeding

Создание класса для работы с заполнением
php artisan make:seeder UserSeeder - создание генератора данных
php artisan db:seed - запуск всех генераторов
php artisan db:seed --class=UserSeeder - запуск генератора по названию класса

Возможно
- еденичное
- массовое
рандомно сгенерированных данных
- в ларавель встроена библиотека генерации тестовых данных Faker
